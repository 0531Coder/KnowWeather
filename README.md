#特点
 * 支持县级、区级天气
 * JobScheduler与JobService的使用
 * 轮询系统定时更新天气
 * 实现很多定制化很大的，对后台任务严苛的系统中(如魅族系统)通知栏常驻，并且轮询系统正常运行，很多下载量千万级的天气应用无法达到此效果
 * 无广告和强制保活拉活的行为
 * ...
